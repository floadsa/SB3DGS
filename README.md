# SB3DGS

### description:
**_SB3DGS_** _- simple bruncell's 3D graphic software  
Created for easy editing of 3D models.  
Currently under active development._
### build: 
```bash
git clone https://github.com//floadsa/SB3DGS.git
cd SB3DGS
cd build
```

```cmake
cmake .. #building
make
```
```bash
./SB3DGS #launch
```
